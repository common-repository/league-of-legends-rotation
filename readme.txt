== Description ==

Plugin shows actually free champions rotation for League of Legends in widget. This plugin displays the current European rotations.

Demo: http://lolrotation.ovh/

Problems

If plugin not display perfectly, use shortcode for display rotation and change width and height: [rotation width="600" height="240"]

If you have some problems, contact with me via email `contact(at)lolrotation.ovh`. More informations on http://lolrotation.ovh/

Donation

If you want individual functions in this plugin contact with me. 

== Installation ==

1. Download the plugin
2. Upload `wp-lol-rotation.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add widget to your theme
